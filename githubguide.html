GitHub is an important tool for developers to be able to improve and send their code out to the world. Here are some important git commands to know

  1. "git status" is a good command to run after you run another git command to make sure you actually did what you intended.
  2. "git add" is what you use when you've made changes to a file and you want to get it ready to commit
  3. "git commit" adds your current file to a Git history, which allows you to look at and use all the different versions of your code
